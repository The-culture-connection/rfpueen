# RFP Queen - Opportunity Matching Platform

A Squarespace-integrated web application for matching fundraising opportunities based on user profiles.

## Features

- **Matching Algorithm**: Filters opportunities by funding types and matches interests
- **Opportunity Cards**: Interactive cards with Apply, Pass, and Save functionality
- **Dashboard**: View applied and saved opportunities
- **Application Management**: Automatic form detection and instructions

## Files

- `explore.html` - Main explore page with matching algorithm
- `dashboard.html` - Main dashboard with navigation
- `dashboard-applied.html` - Applied opportunities page
- `dashboard-saved.html` - Saved opportunities page

## Firebase Collections

- `SAM` - Contracts
- `grants.gov` - Grants
- `grantwatch` - Grants
- `PND_RFPs` - RFPs
- `rfpmart` - RFPs
- `bid` - Bids

## Installation

1. Copy HTML files to Squarespace Code Injection
2. Configure Firebase credentials
3. Set up Firestore collections
4. Deploy to Squarespace pages
